Q1: 在开发包中"演示程序"文件夹下的各个Demo要如何运行？
A1: 请要将各个Demo拷贝到"库文件"文件夹下，即可运行。
	例如：需要运行"演示程序"中的"参数配置"Demo时，将"参数配置"文件夹中的"参数配置.exe"、"IPFileter.ini"
	和"langchn.ini"拷贝至"库文件"文件夹下，运行"参数配置.exe"。

Q2: 如何运行"远程回放"Demo？
A2: 将"远程回放"文件夹下的"远程回放.exe"、"langchn.ini"、"registry.bat"和"DayTimePickSDK.ocx"拷贝到"库文件"文件夹下后，
	需要先运行"registry.bat"脚本，以注册"DayTimePickSDK.ocx"播放控件，再运行"远程回放.exe"即可。

Q1: How to run the demo in the folder of "Demo.exe" correctly?
A1：You should copy the demo to the "DLL" folder. For example, when you want to run the "Parameter Configuration.exe" program,
	you need to copy the "Parameter Configuration.exe" file to the "DLL" folder first, then run the "Parameter Configuration.exe".
	By the way, if you copy the "IPFileter.ini" file as well, you will get the "Parameter Configuration.exe" program running as Chinese,
	so if you want to run the demo as english, DO NOT copy the .ini files to the "DLL" folder. 

Q2: How to run the "Playback.exe" demo?
A2: First of all, copy the "Playback.exe", "registry.bat" and "DayTimePickSDK.ocx" to the "DLL" folder, then run the "registry.bat" to register the
	"DayTimePickSDK.ocx" ocx controler.
